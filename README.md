
# Svelte Todo

[freeseamew](https://github.com/freeseamew)의 [Svelte 강좌](https://freeseamew.gitbook.io/svelte/5./todo-store-3-store)에 나온 미완성 To Do 앱을 마음대로 완성하였습니다.

## 의존성 설치

의존성을 설치합니다.

```bash
cd svelte-todo
npm install
```

## 실행 방법

실행합니다.

```bash
npm run dev
```

`--host 0.0.0.0` 옵션이 필요할 수 있습니다.
